README for VSPEC/test/data/test_reflected

This directory contains data used to test the
`VSPEC.psg_api.get_reflected()` function

The files are as follows:
----------
atm_cmb.rad
    Planet+Star, molecular signatures = ON, R=50
no_atm_cmb.rad
    Planet+Star, molecular signatures = OFF, R=50
hires_cmb.rad
    Planet+Star, molecular signatures = OFF, R=500
no_atm_therm.rad
    Planet, molecular signatures = OFF, R=50
atm_therm.rad
    Planet, molecular signatures = ON, R=50
